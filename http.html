1、各个版本HTTP的区别 HTTP 1.0与1.1的区别： 1)缓存处理 —— HTTP 1.0主要使用header里的If-Modified-Since, Expires作缓存 HTTP 1.1使用更多的缓存策略，E-tag, If-Unmodified-Since, If-Match, If-None-Match 2)带宽优化及网络连接使用 HTTP 1.0中存在浪费带宽的现象 HTTP 1.0中在请求头中引入了 range 头域，它允许只请求资源的某个部分，代码返回206 3）错误通知的管理 HTTP 1.1中增加了24个错误响应状态码， 4）HOST头处理 5）长连接 HTTP 1.1 支持长连接和请求的流水线处理 HTTP 2.0与HTTP 1.x 的区别： 1）新的二进制格式 2）多路复用（连接共享），即每一个request都是用作连接共享机制的，一个request对应一个id, 这样连接上可以有多个request，每个连接的request可以随机的混杂一起 3）header压缩 HTTP 1.x中的header带有大量信息，每次都要重复发送 HTTP 2.0使用encoder来减少传输header大小 4）服务端推送 递归的优化方法： 1）尾递归 2）利用缓存
